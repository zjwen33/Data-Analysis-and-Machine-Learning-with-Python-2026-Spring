🛠️ Developer Data Access (Open Data)

我們秉持開放資料精神，提供唯讀的 MongoDB 連線字串，讓開發者可以直接存取並二次開發本系統的資料庫。

# MongoDB Read-Only Connection String MONGO_URI = "mongodb+srv://:@cluster0.mongodb.net/?retryWrites=true&w=majority"

💡 Python 連線範例:

import pymongo

client = pymongo.MongoClient(MONGO_URI)
db = client['google_maps_reviews_db']
collection = db['reviews']

# 範例：找出所有包含假評論的店家
results = collection.find({"analysis_result.fake_count": {"$gt": 0}})
for doc in results:
    print(doc["place_info"]["title"])
